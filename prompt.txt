## ‚úÖ PROMPT (ENGLISH)

Build a **mobile-first, high-converting SSP (single static page / static site)** website for a **Greek guest house / vacation rental**, targeting locations: **Ofrynio, Kavala, Nea Peramos, and any other Greece locations**.

## Page structure (single page `/`)

### 1) Header (sticky, minimal)

* Logo / house name
* 1 CTA button: ‚ÄúContact‚Äù (scrolls to contact area)
* Language switch optional (EN/GR/BG-ready structure)

### 2) Hero section (top)

Must include:

* **House name** (e.g., ‚ÄúVilla Ammolofi Retreat‚Äù)
* Short tagline (e.g., ‚ÄúSteps from the sea ‚Ä¢ Family-friendly ‚Ä¢ Quiet neighborhood‚Äù)
* 2 CTAs:

  * **View Gallery** (scrolls to gallery)
  * **Contact Owner** (opens a modal or scrolls to contact)
* Key chips (icons): ‚ÄúNear beach‚Äù, ‚ÄúParking‚Äù, ‚ÄúWi-Fi‚Äù, ‚ÄúAir conditioning‚Äù, etc.
* Hero background image (beautiful exterior / seaside vibe)

### 3) Quick highlights block (below hero)

* Distance to beach (e.g., ‚Äú3 min walk to the beach‚Äù)
* Capacity (e.g., ‚ÄúSleeps 6‚Äù)
* Bedrooms/Bathrooms
* Check-in/out times (optional)
* A mini ‚ÄúWhy you‚Äôll love it‚Äù list (3‚Äì5 bullets)

### 4) Gallery section

* Mobile-first gallery (swipeable carousel on mobile, grid on desktop)
* Lightbox on click (tap to zoom)
* Optimize: lazy-loading, blurred placeholder

### 5) About / Location description

* Text: where it is (Ofrynio / Kavala / Nea Peramos etc.)
* Clearly state: **is it close to the beach**, what type of beach, nearby amenities (supermarket, tavernas)
* Add ‚ÄúNearby attractions‚Äù (3‚Äì6 items)

### 6) Amenities section

* Use icon cards
* Group amenities by category (Comfort, Kitchen, Outdoor, Family)

### 7) Contact section + floating buttons

**Floating buttons fixed on the right side** (mobile and desktop):

* WhatsApp
* Viber
* Telegram
* Facebook
* Instagram
* Phone call

Rules:

* Buttons should be stacked vertically on the right.
* On mobile, they should be compact and not cover important content (allow collapse/expand).
* Each button uses the correct deep link format:

  * WhatsApp: `https://wa.me/<number>`
  * Viber: `viber://chat?number=<number>` (and include a fallback link)
  * Telegram: `https://t.me/<username>`
  * Phone: `tel:<number>`
  * Facebook/Instagram: profile URL

Also include a normal contact block with:

* Owner name
* Phone
* Optional email
* ‚ÄúCopy‚Äù buttons for phone/email

### 8) Map section (bottom)

* A map embed (Google Maps iframe is fine for SSP)
* Show approximate pin / exact address (configurable)
* Add a small disclaimer if needed: ‚ÄúLocation shown is approximate‚Äù (optional)

### 9) Footer

* Simple: house name, location, social links
* ‚ÄúInvoice & contract available‚Äù note (if the site is used by an agency)

        function openLightbox(index) {
            currentImageIndex = index;
            updateLightboxImage();
            lightbox.classList.remove('hidden');

                lightbox.classList.remove('opacity-0');

            currentImageIndex = (currentImageIndex - 1 + propertyData.galleryImages.length) % propertyData.galleryImages.length;
